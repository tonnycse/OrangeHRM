# OrangeHRM Manual Testing Project

This project involves manual testing of the OrangeHRM demo sites Login & Apply Leave features. It includes detailed documentation on project details, decision tables, test scenarios, test cases, and bug reports.

## Features
- **Project Details**: Overview of the OrangeHRM demo site and its functionalities.
- **Decision Table**: Logical decision-making matrix for test cases.
- **Test Scenarios and Test Cases**: Comprehensive scenarios and step-by-step test cases for manual testing.
- **Bug Reports**: Documented issues found during testing.

## Project Structure
- **/project-details**: Contains an overview of the project.
- **/decision-table**: Includes the decision table used for deriving test cases.
- **/test-scenarios**: Detailed test scenarios.
- **/test-cases**: Step-by-step test cases with expected and actual results.
- **/bug-reports**: Detailed reports of bugs identified during testing.

## Key Functionalities Tested
- User login and logout
- Apply leave feature
- Validating error handling for incorrect inputs

## Tools Used
- Microsoft Excel for creating test cases and bug reports
- Browser (e.g., Chrome ) for executing tests
